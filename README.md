# TikTok Creator Intelligence System

> NLP-based web app for analyzing TikTok comments and generating 
> actionable content insights for small creators (1k–10k followers).

## Project Overview
This is a university NLP project built with Python and Streamlit.
It analyzes TikTok comments using NLP techniques to help creators
understand their audience and improve their content strategy.

## Core Features
- Sentiment analysis of comments
- Keyword and topic extraction
- Video performance analysis
- Insight and recommendation generation

## Tech Stack
- Python, Streamlit, spaCy, NLTK, scikit-learn, pandas

## Project Status
🚧 In Development — University NLP Course (Spring 2026)

## How to Run
```bash
pip install -r requirements.txt
streamlit run app/main.py
```
